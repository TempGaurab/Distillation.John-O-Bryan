# 📘 Results Dataset Codebook (`results.csv`)

This dataset contains outputs from a dual-agent math evaluation system using the **DeepSeek Reasoner API**.

Each row represents a single math question along with:

- The ground-truth answer
- The solver model's response
- The verifier model’s judgment
- A final correctness flag

## 🔁 System Overview

### 🧠 Agent 1 — Solver

- Model: `deepseek-reasoner`
- Input: Math question
- Output:
  - `agent1_answer` (final answer only)
  - `agent1_reasoning` (chain-of-thought)

### ✅ Agent 2 — Verifier

- Model: `deepseek-reasoner`
- Input:
  - Question
  - Ground-truth answer
  - Agent 1’s answer
- Output:
  - `agent2_verdict`
  - `agent2_justification`

---

## 📊 Columns Description

### `year`

- **Type:** String
- **Description:** Identifier derived from the JSON filename (typically the year of the dataset).
- **Example:** `2024`

---

### `section`

- **Type:** String
- **Description:** Category or section of the question within the dataset.
- **Example:** `freshman_sophomore_test`

---

### `question_id`

- **Type:** String
- **Description:** Unique identifier for each question within a section.
- **Example:** `question_1`

---

### `question`

- **Type:** String
- **Description:** Full text of the math problem.
- **Example:** Find the value of x that satisfies (x-4)/(x-5) = (x+3)/(x+8).

---

### `true_answer`

- **Type:** String
- **Description:** Ground-truth correct answer from the dataset.
- **Format:** Matches required answer format (e.g., integer, reduced fraction).
- **Example:** `17/6`

---

### `agent1_answer`

- **Type:** String
- **Description:** Answer generated by Agent 1 (solver).
- **Constraint:** Must contain only the final answer (no explanation).
- **Example:** `17/6`

---

### `agent1_reasoning`

- **Type:** String
- **Description:** Chain-of-thought reasoning from Agent 1.
- **Purpose:** Debugging, interpretability, and error analysis.
- **Note:** May be empty depending on API response.
- **Example:** Cross-multiply both sides and solve for x...

---

### `agent2_verdict`

- **Type:** Categorical (String)
- **Possible Values:**
- `CORRECT` → Answers are mathematically equivalent
- `INCORRECT` → Answers are not equivalent
- `UNKNOWN` → Unexpected or malformed response
- **Description:** Output from Agent 2 (verifier).

---

### `agent2_justification`

- **Type:** String
- **Description:** Short explanation supporting the verifier’s decision.
- **Example:** Both answers simplify to the same reduced fraction.

---

### `correct_flag`

- **Type:** Integer (Binary)
- **Values:**
- `1` → Agent 1 answer is correct
- `0` → Agent 1 answer is incorrect or needs review
- **Description:** Final correctness indicator derived from `agent2_verdict`.

---

## ⚙️ Evaluation Logic

```text
If agent2_verdict == "CORRECT" → correct_flag = 1
Else → correct_flag = 0


```
